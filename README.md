# Bike Sharing Dashboard

## Setup environment
* conda create --name main-ds python=3.9
* conda activate main-ds
* pip install pandas 
* pip install matplotlib 
* pip install seaborn 
* pip install streamlit 
* pip install babel

## Run steamlit app
streamlit run dashboard_proyek.py
